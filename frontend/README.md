# Frontend - Belanja Aja

Aplikasi web frontend untuk pelanggan.

## Fitur
- Katalog produk
- Keranjang belanja
- Checkout
- Profil pengguna
- Riwayat pembelian

## Teknologi
- HTML5
- CSS3
- JavaScript (ES6+)
- React/Vue.js (opsional)

## Struktur
```
frontend/
├── src/
│   ├── components/
│   ├── pages/
│   ├── styles/
│   └── utils/
├── public/
└── build/
```
