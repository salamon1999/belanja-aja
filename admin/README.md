# Admin Panel - Belanja Aja

Panel administrasi untuk mengelola aplikasi e-commerce.

## Fitur
- Dashboard analytics
- Manajemen produk
- Manajemen kategori
- Manajemen pesanan
- Manajemen pengguna
- Laporan penjualan
- Pengaturan sistem

## Teknologi
- HTML, CSS, JavaScript
- Chart.js untuk grafik
- DataTables untuk tabel data
- Bootstrap untuk UI framework

## Akses
- URL: `/admin`
- Login: Admin credentials required
